# Personal Portfolio Website

This project is built with [Next.js](https://nextjs.org/) and is [hosted using GitHub Pages](https://josephbarkie.com).

The purpose of this project is to highlight my resume and to help further my knowledge in web design & development.

## Development

```bash
npm run dev     # Start dev server at http://localhost:3000
```

## Building & Deployment

```bash
npm run build   # Create production build in build/
npm run deploy  # Build and deploy to GitHub Pages
```

Deployment also runs automatically via GitHub Actions on every push to `main` that includes changes to `src/`, `public/`, `package.json`, or workflow files.
